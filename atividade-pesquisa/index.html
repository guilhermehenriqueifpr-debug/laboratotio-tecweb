Como usar negrito e itálico

Como eu posso usar a tag <b> e a tag <i>?

Para usar negrito, utilize a tag <b> assim:

<b>Texto</b>

Para usar itálico, utilize a tag <i> assim:

<i>Texto</i>

Exemplo prático:
